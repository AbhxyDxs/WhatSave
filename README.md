## WhatSave: WhatsApp Status Saver

A simple application that allows you to save WhatsApp statuses (image and video).

#### Basic Features:
This application is really basic; here its features:
* Android 11 support
* Preview and save files
* Night mode
* No ADs
* No watermark
* It's free!
